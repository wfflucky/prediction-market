## Edit profile

<Steps>
  <Step>
    ### Open settings

    Go to <strong>/settings</strong>.
  </Step>
  <Step>
    ### Update profile fields

    Upload avatar (JPG/PNG/WEBP up to 5 MB), edit email and username, then save.
  </Step>
  <Step>
    ### Preview public page

    Use <strong>View Public Profile</strong> to see how your profile appears to others.
  </Step>
</Steps>

## Username rules

- Usernames are unique.
- Allowed characters: letters, numbers, dot, hyphen.
- Username cannot start or end with dot/hyphen.
